# Snake

A snake game in macOS terminal which is more fun to program than play :stuck_out_tongue_winking_eye:

Up - <kbd>w</kbd>/<kbd>i</kbd>
Down - <kbd>s</kbd>/<kbd>k</kbd>
Left - <kbd>a</kbd>/<kbd>j</kbd>
Right - <kbd>d</kbd>/<kbd>l</kbd>
